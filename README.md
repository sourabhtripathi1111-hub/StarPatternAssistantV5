# Star Pattern Assistant V5 AI

Clean Android source for GitHub Actions APK build.

Included modules:
- Manual Reading Mode
- Direct App Reading menu and screen-capture permission base
- Overlay prediction box
- CSV file that opens in Excel
- Pattern Learning Library
- N-gram prediction engine with confidence and skip signal
- Family prediction: Yellow/Big, Purple/Small, Green/Small, Pink/Big
- AI Pattern Help export prompt for ChatGPT/other AI recheck

Important: direct automatic planet reading needs crop calibration and icon templates from the real game screen for best accuracy. This source includes the architecture/menu/base so the next update can add calibrated icon matching/OCR.
